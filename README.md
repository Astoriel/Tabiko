### features ˚ ༘♡ ⋆｡˚

<details>
<summary><strong>1 Add ur first workspace</strong></summary>

![1st workspaces](screenshot/1st%20workspaces.png)
name it, export it and manage it hehe. you can add and move links between workspaces and add as many as your soul desires ✦

</details>

<details>
<summary><strong>various notes & smart index</strong></summary>

![smart index](screenshot/smart_index.png)
and thanks to smart index, you can find your links across your workspaces based on your notes and/or link titles! you can also save current windows with tabs as links:

![save current window](screenshot/save%20current%20window%20as%20workspace.png)

</details>

<details>
<summary><strong>export ur tabs</strong></summary>

export your tabs in a convenient `.md` format (convenient for text and if you use obsidian)

![export button](screenshot/export.png)
![exported tabs](screenshot/export_ur_tabs.png)

similarly, you can import workspaces with links and notes simply by preserving the format:

![import](screenshot/import.png)

</details>

<details>
<summary><strong>q buddy</strong></summary>

and now you have q buddy! you can give it links and it will save them for you

![buddy.png](screenshot/buddy.png) 
yay! 

</details>
