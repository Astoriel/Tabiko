# Project Status

**Status:** Maintenance mode  
**Snapshot date:** 2026-03-19  
**Primary audience:** users or developers who want a small local Chrome tab/workspace manager.

`Tabiko` is a lightweight browser extension for tab workspaces, notes, import/export, and local organization. It is intentionally small and personal in tone.

## Current Confidence

- Working: extension UI, workspace organization, notes, search/index behavior, import/export flow.
- Planned: clearer install docs, manual smoke checklist, changelog, better automated regression coverage.
- Not claimed: active Chrome Web Store listing, synced cloud workspace service, enterprise tab management.
